These are the modules for your game.<br><br>
The idea is that if you create your own html file (or use something like Google Sites) you can customize your layout for how you want to play the game.
In other words, you can add these as **iframes** into that HTML file and these will automatically interact with each other because the Javascript files in the js folder handles the interconnections.
<br><br>
Think of these as iframes. Select the raw output and copy that link into iframes on your Google Site or wherever.
